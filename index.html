SNIPPET..

Save, recall, add or delete code. The choice is yours to control.

This website utilizes the user's local storage within their browser to save snippets of code for later use. Simply type it in the text-box provided, name the snippet and click save.

The website also uses google-prettify for keyword highlightin for clearer reading, the responsive design of the website allows the user save code from their mobile phone.
